## 制作の背景と目的

### 1. 現状の課題と制作のきっかけ
母親が運営するダンス教室では、既存のノーコードツールで作成されたWebサイトが稼働していましたが、スマートフォンでの視認性や情報の整理に課題があり、
新規集客や既存生徒への案内がスムーズに行えていない状態でした。
そこで、ユーザーが本当に欲しい情報へ迷わずアクセスできるよう、既存サイトの課題を分析し、1からフルスクラッチでのリニューアルを企画しました。

### 2. サプライズ提案から実務的なブラッシュアップへ
まずは事前の要望を踏まえたプロトタイプを自作し、サプライズという形で母親へ提示。その後、実際の運用に合わせた以下の要件定義や改善プロセスを経験しました。

* ヒアリングと内容訂正: 母親からのフィードバックを基に、クラスの難易度表示の視覚化や、スケジュールの色分けなど、現場のニーズに合わせたデザイン・コードの修正を行いました。
* ユーザー目線での導線テスト: 実際に公開環境で各種申し込みフォームやLINEコピー機能の使い勝手を自らテストし、ユーザーがスマートフォンから迷わず、かつ最小限のタップ数でアクションを起こせるよう導線を最適化しました。

### 3. 本プロジェクトのゴール
「高松市の地元のユーザーがスマホで見たときに、一番信頼できて通わせたくなるサイト」を目指し、デザインの美しさだけでなく、実務におけるUI/UXの向上と集客力の最大化を目的としています。
